# Kakuro Playground: Where Numbers Meet Logic

Kakuro Playground is an innovative web application that brings the classic Japanese puzzle game Kakuro to the digital age. Built with modern web technologies, it offers a unique feature that allows players to upload images of Kakuro puzzles and play them directly in their browser.

## What is Kakuro?

Kakuro, also known as Cross Sums, is a logic puzzle that combines elements of sudoku and crosswords. Players fill in cells with numbers that sum up to specified totals while following these rules:

- Only numbers 1-9 can be used
- No number can be repeated in a sum
- Empty cells must be filled with a single digit
